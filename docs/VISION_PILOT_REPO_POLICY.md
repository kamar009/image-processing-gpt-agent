# Политика хранения артефактов пилота Vision

Цель: не раздувать git большими наборами картинок и сырыми CSV, но сохранять воспроизводимость и ссылки для решения по провайдеру.

## Рекомендации

1. **В репозитории** — методология, шаблоны манифестов, итоговые выводы: [VISION_PROVIDER_DECISION.md](VISION_PROVIDER_DECISION.md), при необходимости короткие MD-резюме без бинарников.
2. **Локально или в объектном хранилище / артефактах CI** — полные наборы `test_datasets/vision_pilot/**/*.jpg` (и аналоги), большие CSV пилота.
3. Каталог **`test_datasets/vision_pilot/`**: в `.gitignore` игнорируются типичные расширения изображений; **README** в этой папке остаётся в git и описывает структуру и откуда взять файлы.
